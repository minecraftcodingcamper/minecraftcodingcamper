Dynamic Minecraft Structure Tutorial

Welcome to the Minecraft coding tutorial! Follow the steps below to create a dynamic structure in Minecraft Education. Each step includes a code snippet you can copy and paste into MakeCode.

Step 1: Register a Chat Command

First, let’s create a chat command so Minecraft knows when to run our code. Add the following code:

def on_on_chat():
    pass

player.on_chat("run", on_on_chat)

Step 2: Build a Pillar

Now, let’s create a function to build a single pillar. Add this code inside your on_on_chat function:

def build_pillar(x, z):
    blocks.place(POLISHED_BLACKSTONE, pos(x, 0, z))
    blocks.place(BLACKSTONE_WALL, pos(x, 1, z))
    blocks.place(OAK_FENCE, pos(x, 2, z))
    blocks.place(POLISHED_BLACKSTONE, pos(x, 3, z))

Step 3: Add Multiple Pillars

Let’s place four pillars to form the corners of a square:

build_pillar(1, 0)
build_pillar(11, 0)
build_pillar(1, 10)
build_pillar(11, 10)

Step 4: Connect the Pillars with Glass

Create a function to connect the pillars with glass:

def connect_with_glass(x1, z1, x2, z2):
    if x1 == x2:
        z3 = min(z1, z2) + 1
        while z3 < max(z1, z2):
            blocks.place(GLASS, pos(x1, 3, z3))
            z3 += 1
    elif z1 == z2:
        x3 = min(x1, x2) + 1
        while x3 < max(x1, x2):
            blocks.place(GLASS, pos(x3, 3, z1))
            x3 += 1

Step 5: Fill the Center

Finally, fill the center with BIG_DRIPLEAF:

blocks.fill(
    BIG_DRIPLEAF,
    pos(2, 0, 1),
    pos(10, 1, 9),
    FillOperation.REPLACE)
